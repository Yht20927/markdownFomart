# md-table-to-image Skill

Render Markdown tables as PNG/JPG images. Uses Puppeteer for pixel-perfect rendering. Same approach as md-to.com.

## Usage

```bash
node index.js convert -i input.md -o output.png
node index.js convert -i input.md -o output.jpg
```
