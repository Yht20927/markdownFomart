# md-to-image Skill

Convert Markdown to high-resolution PNG/JPEG image. Uses Puppeteer for pixel-perfect rendering — same approach as md-to.com's image export.

## Usage

```bash
node index.js convert -i input.md -o output.png
node index.js convert -i input.md -o output.jpg -w 1200
```
