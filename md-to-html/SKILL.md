# md-to-html Skill

Convert Markdown to clean HTML5 with code highlighting and KaTeX math rendering. Same pipeline as md-to.com.

## Usage

```bash
node index.js convert -i input.md -o output.html
```

Output is a complete HTML5 document with syntax-highlighted code blocks and rendered math formulas.
